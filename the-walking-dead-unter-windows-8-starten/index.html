The Walking Dead unter Windows 8 starten

Ich bin ein großer Fan von The Walking Dead, sowohl von der TV-Serie als auch vom gleichnamigen Spiel; das großartige und packende Adventure, nicht der minderwertige mittelmäßige First-Person-Shooter.

Eines der Dinge, die mir am Spiel sehr gut gefallen haben, ist die schrittweise Veröffentlichung in mehreren Episoden. Dadurch erstreckt sich die Spielzeit über einen relativ langen Zeitraum und man kann sich immer wieder auf die nächste Episode freuen, genau wie bei der Serie. Das führte allerdings auch dazu, dass ich zwischenzeitlich mein Betriebssystem gewechselt habe, von Windows 7 zu Windows 8. Eigentlich kein Problem, sollte man meinen.

Das Problem

Gewissenhaft wie ich bin, habe ich vor dem OS-Upgrade noch meine Spielstände gesichert. Nur auf Steam wollte ich mich in diesem Fall dann doch nicht verlassen. Nach dem Upgrade also die alten Spielstände wiederhergestellt und dann kam der erste Start des Spiels. Genauer gesagt, er kam nicht. Stattdessen gab’s eine böse Überraschung. Das Spiel wollte einfach nicht starten. Da half auch kein Kompatibilitätsmodus.

Ursachenforschung

Nachdem der erste Schock überwunden war, habe ich dann erst mal einen Blick auf die Problemdetails geworfen. Es gab also Knatsch mit der dinput8.dll, irgendsoeinem Bestandteil von DirectX.

Aus weiser Voraussicht habe ich auf die automatische Problembehandlung von Windows verzichtet und stattdessen gleich Google befragt. Wie sich zeigte, ist das Problem wohlbekannt und tritt auf, wenn man die Steam-Variante von The Walking Dead unter Windows 8 startet und gleichzeitig ein Gamepad angeschlossen hat. Zugegeben, darauf wäre ich, ohne Hilfe, wohl nicht so schnell gekommen. Erfreulicherweise gibt es zwei simple Lösungen.

Lösung 1 - Weg mit dem Gamepad

So einfach kann es sein; das Gamepad abstöpseln und schon läuft wieder alles. Ist natürlich auf Dauer etwas unschön, wenn man ständig das Gamepad an- und abstecken muss, je nachdem, was man gerade spielt. Richtig problematisch wird diese Lösung jedoch, wenn man The Walking Dead mit Gamepad spielen will. Glücklicherweise gibt es noch eine weitere Lösung, die auch das möglich macht.

Lösung 2 - Austauschen der dinput8.dll

Unter Windows 7 lief noch alles wunderbar. Was liegt also näher, als die Version der dinput8.dll auszuprobieren, die auch unter Windows 7 verwendet wird. Und tatsächlich, damit funktioniert auch unter Windows 8 wieder alles wie gehabt.

Bleibt die Frage, woher man diese DLL bekommen soll. In allen Lösungen, die ich im Netz zu dem Problem gefunden habe, wurde auf diverse (dubiose) Websites verwiesen, die alle möglichen DLLs anbieten. Zum Teil wurde auch auf selbst hochgeladene Dateien verlinkt. Das ist sicher bequem, setzt aber auch voraus, dass man der jeweiligen Website auch wirklich vertrauen kann.

Für die Faulen Vertrauensseligen unter euch biete ich hier eine weitere dubiose Quelle für die DLL an; von mir persönlich aus einem originalen Windows-7-Installations-DVD-Image extrahiert.

Download: dinput8.dll
(142 KB, v6.1.7600.16385, MD5: 5E08AC958BE05247FF1539E0D1CE7905, SHA1: 285C3DC75669DFBCF72AD9FCAC4891E67B5FE00F)

Vertrauen ist gut, Kontrolle ist besser

Auf der sicheren Seite ist man, wenn man sich die DLL einfach aus einer bestehenden Windows-7-Installation klaut oder auch direkt von der Installations-CD. Alternativ funktioniert das Ganze übrigens auch mit Windows XP SP3 (frühere Versionen ungetestet). Nimmt man die Datei aus einem bestehenden System, kann man sie einfach kopieren. Die gesuchte Datei findet man jeweils unter den folgenden Pfaden:

  • Windows XP: C:\Windows\System32\dinput8.dll
  • Windows 7 (x86): C:\Windows\System32\dinput8.dll
  • Windows 7 (x64): C:\Windows\SysWOW64\dinput8.dll

Holt man sie direkt von der CD bzw. deren Image, benötigt man noch einen Packprogramm (z.B. 7-Zip). DINPUT8.DL_ und install.wim sind jeweils Archive. D steht für den Laufwerksbuchstaben unter dem das Installationsmedium zu finden ist.

  • Windows XP - CD: D:\I386\DINPUT8.DL_\dinput8.dll
  • Windows 7 (x86) - DVD: D:\sources\install.wim\1\Windows\System32\dinput8.dll
  • Windows 7 (x64) - DVD: D:\sources\install.wim\1\Windows\SysWOW64\dinput8.dll

Der letzte Schritt

Hat man die DLL erst einmal, muss sie nur noch ins Installationsverzeichnis von The Walking Dead gelegt werden. Standardmäßig wird das Spiel systemabhängig unter den folgenden Pfaden installiert:

  • 32-Bit-System: C:\Program Files (x86)\Steam\SteamApps\common\The Walking Dead\
  • 64-Bit-System: C:\Program Files\Steam\SteamApps\common\The Walking Dead\

Alternativ kann man auch einfach in seiner Steam-Library den Eintrag für The Walking Dead mit der rechten Maustaste anklicken und dann Properties auswählen. In dem sich öffnenden Fenster findet man dann unter dem Reiter LOCAL FILES den Button BROWSE LOCAL FILES…, der einen direkt zum Installationsordner führt.

Startet man das Spiel nun, wird automatisch die ältere Datei geladen und nicht die Inkompatible aus dem Systemverzeichnis. So einfach ist es also The Walking Dead unter Windows 8 zu spielen. ;-)
